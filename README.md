<h2 id="start">▶ 시작하기</h2>

<br>

**프로젝트 설치**

```bash
git clone https://github.com/marshot9420/marshot-frontend
npm install -g yarn
```

<br>

**환경변수 설정**

> `.env.development` 파일에 환경변수를 넣어<br> > **루트 디렉토리**에 위치시켜야 애플리케이션이 정상적으로 동작합니다.
> 👉 [.env.template](.env.template)

**환경변수 설명**
| 키 | 설명 |
| --------------------------- | ------------------------------------------ |
| `PORT` | 포트 번호 |

<br>

**패키지 설치 및 실행**

```bash
yarn            # package.json dependencies 설치
yarn dev  # 개발 모드로 실행
```
